---
title: "Authentication"
description: "In order to be able to use these APIs you will need to create a user profile with AASAAN. Once you have created a user profile, you will need to work with us to get the necessary authorization key. This authorization key is needed for every subsequent API calls that you make which needs to be provided via x-api-key header."
---

<Warning>
You can add or revoke secret keys if the security of any key is compromised.
</Warning>

You have to pass the api-key as a header in every call.
